# Kumbara-Kala

GenAI-Powered Heritage Branding app for promoting traditional clay artisan products using Android Studio, Kotlin, and Jetpack Compose.

---

## 📱 Features

- Display traditional clay products
- Generate AI-inspired heritage stories
- Professional product cards
- Eco-friendly awareness
- Traditional pottery promotion
- Interactive Android UI

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Android Studio
- Material Design

---

## 📦 Products Included

- Clay Pot
- Oil Lamp
- Clay Bottle

---

## 🚀 Future Enhancements

- WhatsApp sharing
- Firebase integration
- Artisan profile page
- AI-generated images
- Multi-language support
